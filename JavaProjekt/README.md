# Snake with Mozart


## Technologies
* Java
* Swing

## Features 
* High score save
* Music choice
* Game pausing

## Installation and configuration 
```git
git clone https://github.com/maciekpop/dotNetJava.git 
cd dotNetJava/
cd JavaProjekt/
cd Snake/
run application
```

* Moving: WASD
* Pausing: Space
* Getting back to Menu: Esc
